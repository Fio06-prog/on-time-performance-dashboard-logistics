# Dataset Description

## Data Source
The dataset used in this project was obtained from **Kaggle** and contains logistics delivery event records.

## Dataset Overview
- Time period: **January 2022**
- Event types: **Pickup** and **Delivery**
- Granularity: Individual delivery events
- Geographic level: State-level locations
- Dataset size: Large operational dataset (full file not included in this repository)

## Key Fields
- Event ID
- Event Type (Pickup / Delivery)
- Scheduled Datetime
- Actual Datetime
- On-time Flag
- Location State

## Data Preparation
- Initial data cleaning and preparation were performed using **Google Sheets**.
- Invalid or incomplete records were excluded from KPI calculations.
- Derived fields (e.g., on-time indicators) were used to support performance analysis.

## Data Availability
The full dataset is **not included** in this repository due to size and licensing considerations.

Anyone interested in reproducing the analysis can obtain the dataset directly from Kaggle and follow the documented analytical approach.

## Disclaimer
This dataset is used for **educational and portfolio purposes only**.  
All insights are illustrative and do not represent real operational performance of any specific company.
